# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: be respectful, assume good faith, and keep discussion focused on the
specification and its improvement. Harassment or exclusionary behaviour is not
tolerated. Maintainers may remove, edit, or reject contributions and comments
that violate this code.

To report a concern, contact the maintainer listed in `README.md`.
